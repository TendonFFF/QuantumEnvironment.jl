# Introduction

QuantumEnvironment.jl provides environment models and approximation utilities for open quantum systems in Julia.

## Installation

```julia
using Pkg
Pkg.add(url="https://github.com/TendonFFF/QuantumEnvironment.jl")
```

## Development Docs Build

```julia
using Pkg
Pkg.activate("docs")
Pkg.instantiate()
include("docs/make.jl")
```
